# Markov Word Generator
A simple stochastic word generator based on the standard Markov text generation scheme using Unix dictionary as a training set.

## Requirements
- uses a Unix dictionary as a trainig set
-Generates words between 5 and 10 letters in length
-Differentiate between beginning letters, ending letters and middle letters
-Generates words in a loop after each press of the "enter" key util a non-empty input is received
